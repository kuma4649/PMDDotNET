# PMDDotNET  
  PMDの.NET版です。  
  
[概要]  
  PMDを.NET版に移植したものです。  
  
[機能、特徴]  
 今のところコンパイラ(mc.exe相当)のみ  
 ドライバも移植中。  
  
[必要な環境]  
 ・Windows7以降のOSがインストールされたPC  
 ・テキストエディタ  
 ・気合と根性
